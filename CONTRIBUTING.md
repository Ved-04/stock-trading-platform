# Contributing to Stock Trading Platform

Thank you for your interest in contributing! Here are the guidelines to follow:

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/yourusername/stock-trading-platform.git`
3. Create a new branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit with clear messages: `git commit -m "Add description of changes"`
6. Push to your branch: `git push origin feature/your-feature-name`
7. Open a Pull Request

## Code Standards

- Use consistent code formatting
- Follow the existing project structure
- Add comments for complex logic
- Test your changes before submitting

## Reporting Issues

- Check if the issue already exists
- Provide clear description and steps to reproduce
- Include error messages and logs if applicable
- Mention your environment (OS, Node version, etc.)

## Pull Request Process

1. Update the README.md if needed
2. Include a clear description of changes
3. Link any related issues
4. Request review from maintainers
5. Address any feedback

## Code of Conduct

- Be respectful and constructive
- Welcome diverse perspectives
- Focus on the code, not the person
- Help others learn and grow

Thank you for contributing!
